GSMaiMai - Tape Emulation Plugin
================================

A vintage tape saturation / wow-flutter / lo-fi effect plugin.
Universal Binary (Apple Silicon + Intel Rosetta).

Formats: VST3, Audio Unit (AU)
Hosts: Cubase, Logic Pro, Ableton Live, Studio One, Reaper, etc.

INSTALL
-------
Double-click "install.command"

If macOS blocks it ("cannot be opened"):
  1) Right-click install.command → Open → Open
  or
  2) Terminal → run: bash install.command

Then restart your DAW.

UNINSTALL
---------
Double-click "uninstall.command"

INSTALL LOCATIONS
-----------------
VST3: ~/Library/Audio/Plug-Ins/VST3/GSMaiMai.vst3
AU:   ~/Library/Audio/Plug-Ins/Components/GSMaiMai.component

PARAMETERS
----------
Wow Rate/Depth       - Slow tape speed warble (pitch+speed coupling)
Flutter Rate/Depth   - Fast tape flutter
Drift                - Long-term random speed drift
Scrape               - High-freq scrape flutter
Saturation / Drive   - Tape saturation amount
LP / HP              - Tape bandwidth limiting
Hiss                 - Tape noise
Mix / Output         - Wet/dry and output gain
